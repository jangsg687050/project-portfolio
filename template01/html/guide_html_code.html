HTML 작성규칙

HTML 기본규칙

들여쓰기

  • 1탭 들여쓰기 하며, 1탭의 크기는 공백 4칸으로 설정한다.

빈줄

  • 가독성을 고려하여 빈 줄을 추가한다. 단, 1줄을 초과하지 않는다.

주석

  • 주석 사용시 날짜, 설명을 정확히 작성한다. (예: YYYY.MM.DD 이미지 수정)
  • 끝의 표시는 //(띄어쓰기) 로 구분한다.
  • 참고주석은 앞에 표시한다. (예: D://)
<!-- 로고 -->
<h1>
    <a href="#로고">로고</a>
</h1>
<!-- // 로고 -->
    
<!-- D:// 완료시 disable 추가 -->
<p class="disable">참고주석 : 시작과 끝이 없고, 전달내용을 설명한다.</p>
    
<!-- YYYY.MM.DD 설명 삭제, 추가, 수정 -->
<p>기록주석 : 수정영역의 범위구분이 필요한 경우 닫기주석을 사용한다.</p>
<p>기록주석 : 수정영역의 범위구분이 필요한 경우 닫기주석을 사용한다.</p>
<!-- // YYYY.MM.DD 설명 삭제, 추가, 수정 -->

<p>기록주석 : 수정영역이 간단한 경우 시작과 끝이 없고 인라인으로 작성할 수 있다.</p> <!-- // YYYY.MM.DD 설명 삭제, 추가, 수정 -->

특수기호

  • 특수기호는 Entity name을 사용하여 entity 코드로 변환한다.
표기법
Character < > & " ©
Entity Name &nbsp; &lt; &gt; &amp; &quot; &copy;
* 참고: HTML Entities

HTML input type

  • input type은 모바일 사용성을 고려하여 반드시 작성항목에 맞게 사용한다.
<input type="text" /> <!-- 일반 텍스트 -->
<input type="tel" /> <!-- 전화번호 -->
<input type="email" /> <!-- 이메일 -->
<input type="password" /> <!-- 비밀번호 -->
<input type="url" /> <!-- url -->
<input type="search" /> <!-- 검색 -->
<input type="del" /> <!-- 삭제 -->
* 참고: 모바일에서의 input type 속성 비교보기

오픈그래프

오픈그래프 사용

  • 이미지 사이즈 최소 600*315, 권장 사이즈 1200*630
<meta property="og:site_name" content="웹사이트 이름">
<meta property="og:title" content="웹사이트 콘텐츠 제목" />
<meta property="og:description" content="웹사이트 콘텐츠 설명" />
<meta property="og:image" content="웹사이트 제목 썸네일 이미지" />
<meta property="og:url" content="웹사이트 URL" />
